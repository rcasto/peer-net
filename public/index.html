PeerNet - Peer Connection without a server

How this works

If you want to accept a connection from someone:

  1. Paste offer received into the text area and accept it
  2. Send the answer back to who sent you the offer, they will need to accept it
  3. You should now be connected!

If you want to initiate a connection:

  1. Create an offer
  2. Send the offer to who you want to connect to, they will send you back an answer
  3. Paste the answer you receive into the text area and accept it
  4. You should now be connected!
